# Inicialización de repositorio Git

1. Abre una terminal en la raíz del proyecto.
2. Ejecuta los siguientes comandos:

```bash
git init
git add .
git commit -m "Proyecto inicial: plataforma de gestión académica y organizacional"
```

Esto creará el repositorio y realizará el primer commit con todos los archivos actuales.

---

> **Recuerda:** Antes de subir a un repositorio remoto (GitHub, GitLab, etc.), crea el repositorio en la plataforma elegida y sigue las instrucciones para agregar el remoto y hacer push.
